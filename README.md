# Modern Responsive Website

A simple, responsive website built with HTML, CSS, and JavaScript. This project demonstrates modern web development practices with a clean, accessible design.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Mobile Navigation**: Hamburger menu for small screens
- **Interactive Contact Form**: Client-side validation and success feedback
- **Semantic HTML5**: Uses proper semantic elements for better accessibility
- **Modern CSS**: Utilizes CSS variables, Grid, and Flexbox
- **Accessible**: ARIA labels and keyboard-friendly navigation

## Files Included

- `index.html` - Main HTML structure with semantic elements
- `styles.css` - Responsive CSS with modern layout and theming
- `script.js` - JavaScript for mobile menu and form validation
- `README.md` - This file with project documentation

## How to View Locally

1. Clone or download this repository to your local machine
2. Navigate to the project directory
3. Open `index.html` in your web browser:
   - **Option 1**: Double-click the `index.html` file
   - **Option 2**: Right-click `index.html` and select "Open with" your preferred browser
   - **Option 3**: Drag and drop `index.html` into an open browser window

That's it! No build tools or server required - this is a static website.

## Browser Compatibility

This website works with all modern browsers:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Sections

The website includes the following sections:

1. **Header/Navigation**: Sticky navigation bar with mobile menu toggle
2. **Hero Section**: Welcome message with call-to-action button
3. **Services Section**: Three feature cards showcasing different services
4. **Contact Section**: Contact form with validation
5. **Footer**: Copyright and links

## Form Validation

The contact form includes validation for:
- Name (required, minimum 2 characters)
- Email (required, valid email format)
- Message (required, minimum 10 characters)

## Technical Details

- **No dependencies**: Pure HTML, CSS, and JavaScript
- **CSS Variables**: For easy theming and maintenance
- **CSS Grid**: For responsive feature cards layout
- **Flexbox**: For flexible component layouts
- **Mobile-first approach**: Responsive breakpoints at 768px and 1024px
