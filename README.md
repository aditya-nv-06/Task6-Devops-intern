# Responsive Portfolio Website

A minimal, responsive static website built with HTML5, CSS3, and vanilla JavaScript. Features a modern design with mobile navigation, interactive elements, and client-side form validation.

## Files Included

- **index.html** - Main HTML file with semantic structure, responsive layout, and accessible navigation
- **styles.css** - Responsive CSS with CSS variables, mobile-first design, and accessible styling
- **script.js** - JavaScript for mobile menu toggle, form validation, and dynamic footer year
- **README.md** - This file with setup and deployment instructions

## View Locally

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build tools or dependencies required - it's pure HTML, CSS, and JavaScript

## Features

- ✅ Responsive design that works on mobile, tablet, and desktop
- ✅ Accessible hamburger menu for mobile navigation
- ✅ Client-side form validation with user feedback
- ✅ Semantic HTML5 elements for better SEO and accessibility
- ✅ CSS Grid for responsive feature cards
- ✅ No dependencies - pure vanilla JavaScript

## GitHub Pages Deployment

To publish this website using GitHub Pages:

### Option 1: Deploy from main branch
1. Go to your repository Settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the `main` branch
4. Select `/ (root)` as the folder
5. Click "Save"
6. Your site will be published at `https://<username>.github.io/<repository-name>/`

### Option 2: Deploy from gh-pages branch
1. Create a new branch called `gh-pages`
2. Push your files to the `gh-pages` branch
3. Go to repository Settings → Pages
4. Under "Source", select the `gh-pages` branch
5. Click "Save"
6. Your site will be published at `https://<username>.github.io/<repository-name>/`

It may take a few minutes for your site to be published. You'll see a notification with your site URL once it's live.
